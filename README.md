#  Flipkart Clone (React + Redux + Tailwind CSS)

A fully responsive Flipkart-like e-commerce application built using **React**, **Redux Toolkit**, **React Router**, and **Tailwind CSS**. The app fetches product data from the **MongoDB** and mimics 
core features of a modern e-commerce platform, like product listing, search, filters, cart management, and product details.

---

##  Features

- ✅ Product Listing Page
- ✅ Product Categories Filter
- ✅ Add to Cart / Remove from Cart
- ✅ Cart Page with Price Summary
- ✅ Product Detail Page
- ✅ Search Product
- ✅ Responsive UI (Mobile, Tablet & Desktop)
- ✅ Redux Toolkit for State Management
- ✅ Data from Mongo DB
- ✅ Tailwind CSS Styling

##  Installation & Setup

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone https://github.com/Anant5555-AI/Flipkart

# Navigate into the project folder
cd Flipkart

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
